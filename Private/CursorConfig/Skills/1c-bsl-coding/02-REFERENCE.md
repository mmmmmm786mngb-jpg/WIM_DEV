## Документирующий комментарий

```bsl
// Описание: Вычисляет сумму с НДС по ставке.
// Параметры:
//   СуммаБезНДС - Число - сумма без НДС
//   СтавкаНДС   - Число - ставка в процентах (например, 20)
// Возвращаемое значение:
//   Число - сумма с НДС
// Пример:
//   СуммаСНДС = РассчитатьСуммуСНДС(1000, 20); // 1200
Функция РассчитатьСуммуСНДС(СуммаБезНДС, СтавкаНДС)
    Возврат СуммаБезНДС * (1 + СтавкаНДС / 100);
КонецФункции
```
