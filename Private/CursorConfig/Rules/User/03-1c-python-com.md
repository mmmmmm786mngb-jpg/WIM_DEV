## Инициализация и подключение:
```python
pythoncom.CoInitialize()
com = win32com.client.Dispatch("V83.COMConnector")
conn = com.Connect("Srvr='localhost';Ref='база';")
# В конце: pythoncom.CoUninitialize()
```

## КРИТИЧЕСКИ ВАЖНО - Правильные типы объектов:

**ПРАВИЛЬНО:**
- СправочникМенеджер - для создания/чтения элементов справочника
- ДокументМенеджер - для работы с документами
- Запрос - для выполнения SQL-запросов

**НЕПРАВИЛЬНО (НЕ работает):**
- СправочникОбъект - Property can not be set
- conn.Справочники() - Член группы не найден
- conn.Выполнить() - Connect.Выполнить (не доступен)
- conn.Вычислить() - Connect.Вычислить (не доступен)

## Создание элемента справочника (ОБЯЗАТЕЛЬНЫЙ порядок):
```python
# 1. Получить менеджер
manager = conn.NewObject("СправочникМенеджер.ИмяСправочника")

# 2. Создать элемент
element = manager.СоздатьЭлемент()

# 3. Заполнить свойства
element.Наименование = "Значение"

# 4. Записать
element.Записать()
```

## Получение метаданных:

```python
metadata = conn.Метаданные
```

## Выполнение запросов:
```python
# Простой запрос
query = conn.NewObject("Запрос")
query.Текст = "ВЫБРАТЬ 1 КАК Один"
result = query.Выполнить()
result_data = result.Выгрузить()
value = result_data[0].Один

# Запрос с выборкой
query = conn.NewObject("Запрос")
query.Текст = "ВЫБРАТЬ ПЕРВЫЕ 10 Наименование ИЗ Справочник.Номенклатура"
result = query.Выполнить()
result_data = result.Выгрузить()

for row in result_data:
    print(row.Наименование)

# Альтернатива: через Выбрать()
selection = result.Выбрать()
while selection.Следующий():
    print(selection.Наименование)
```

## Таблица значений как параметр запроса:
```python
# Создание таблицы значений
value_table = conn.NewObject("ТаблицаЗначений")
columns = value_table.Колонки
type_string = conn.NewObject("ОписаниеТипов", "Строка")
columns.Добавить("Код", type_string)

# Добавление строк
row = value_table.Добавить()
row.Код = "001"

# Запрос с табличным параметром
# ВАЖНО: ПОМЕСТИТЬ должен быть ПЕРЕД ИЗ, не в начале запроса!
query = conn.NewObject("Запрос")
query.Текст = """
ВЫБРАТЬ
    ТаблицаЗначений.Код,
    ТаблицаЗначений.Наименование
ПОМЕСТИТЬ ВременнаяТаблица
ИЗ
    &ТаблицаЗначений КАК ТаблицаЗначений;

    ВременнаяТаблица.Код КАК Код,
    ВременнаяТаблица.Наименование КАК Наименование
ИЗ
    ВременнаяТаблица КАК ВременнаяТаблица
"""

# Установка параметра и выполнение
query.УстановитьПараметр("ТаблицаЗначений", value_table)
result = query.Выполнить()
result_data = result.Выгрузить()

# Для JOIN двух таблиц значений используйте ВЫРАЗИТЬ для приведения типов
query2.Текст = """

ВЫБРАТЬ
    ВЫРАЗИТЬ(Коды.Код КАК Строка(10)) КАК Код
ПОМЕСТИТЬ ВременнаяТаблица2
ИЗ &Коды КАК Коды;
```

## Удаление элементов справочника:
```python
manager = getattr(conn.Справочники, "ИмяСправочника")
find = getattr(manager, "НайтиПоНаименованию", None)
ref = find("ИмяЭлемента")

if not ref.Пустая():
    ref.ПолучитьОбъект().Удалить()
```

## Вызов процедур общих модулей:
```python

# Вызвать процедуру
```

## Ключевые различия API:
- Для СОЗДАНИЯ: `manager = conn.NewObject("СправочникМенеджер.Имя")`
- Для УДАЛЕНИЯ: `manager = getattr(conn.Справочники, "Имя")`
- Для ВЫЗОВА ПРОЦЕДУР: `getattr(conn, "ИмяМодуля") + модуль.Процедура()`
- Для ОБРАБОТОК: `getattr(conn.Обработки, "Имя") + manager.Создать() + obj.Функция()`

## Типичные ошибки:
1. "Property can not be set" -> используйте СправочникМенеджер
2. "Тип не определен" -> проверьте регистр и правильность имени
3. COM не подключается -> откройте 1С вручную (для файловой базы)
4. "Содержимое объекта данных может быть выбрано только во временную таблицу" -> используйте ПОМЕСТИТЬ перед ИЗ
5. "Неверные параметры в операции сравнения" -> используйте ВЫРАЗИТЬ для приведения типов

## Помнить ВСЕГДА:
- Используйте СправочникМенеджер, НЕ СправочникОбъект
- Сначала СоздатьЭлемент(), потом заполнять поля
- Обязательно Записать() после изменений
- Для запросов: `query.Текст = "..."` затем `query.Выполнить()`
- Для таблиц значений: используйте `value_table.Колонки` (не Колонцы)
- Для табличных параметров: ПОМЕСТИТЬ должен быть ПЕРЕД ИЗ
- Для JOIN таблиц значений: используйте ВЫРАЗИТЬ для приведения типов
- Для обработок: ОБЯЗАТЕЛЬНО указать `App='PyCOM'` и `Locale=ru_RU` в строке подключения
- Для общих модулей: модуль должен иметь свойство "Внешнее соединение"
