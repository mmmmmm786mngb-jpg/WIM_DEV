# WIM_DEV

Репозиторий для разработки на 1С 8.3. Несколько базовых конфигураций; под каждой — папка **Проекты**, в ней создаются проекты.

## Папки верхнего уровня

| Папка | Назначение |
|-------|------------|
| [bases/](bases/) | Базовые конфигурации и проекты (по одной или нескольким базам) |
| [Черновики/](Черновики/) | Черновики, наброски, незавершённые материалы |
| [КодРевью/](КодРевью/) | Отчёты и материалы ревью кода |

## Базы и исходники

Реальные базы в **C:\\1c\\Cursor_1c\\WORK** (SBB_Broker, SBB_Depo, Wim_Du, WIM_FIn, Wim_Mo). В репозиторий не копируются.

В каждой базе в [bases/](bases/):
- **source-path.txt** — путь к каталогу с исходниками (для скриптов и проектов)
- **Проекты/** — папка для проектов на этой базе

**Общие проекты** (несколько баз): [bases/Общие/Проекты/](bases/Общие/Проекты/). Пути к базам — в папках баз, файл source-path.txt; в проекте можно указать список баз в **bases.txt**.

## Создание проектов

- **Одна база:** создавайте в **bases/<база>/Проекты/**. Шаблон: [bases/SBB_Broker/Проекты/example/](bases/SBB_Broker/Проекты/example/).
- **Несколько баз:** создавайте в **bases/Общие/Проекты/**. Шаблон: [bases/Общие/Проекты/example/](bases/Общие/Проекты/example/).

В проекте: Расширения, Документация, Тестирование, Скрипты. Путь к коду базы — в **source-path.txt** папки базы (для общих — ../../<база>/source-path.txt).
