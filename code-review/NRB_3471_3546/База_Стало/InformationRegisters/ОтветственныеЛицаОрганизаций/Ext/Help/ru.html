Регистр сведений "Ответственные лица организаций"

Регистр сведений "Ответственные лица организаций" используется для сохранения истории изменений данных об ответственных лицах организаций.

Реквизиты

Структурная единица - организация, где данное физическое лицо является ответственным лицом. Выбирается из справочника "Организации".

Ответственное лицо - роль физического лица в организации: Руководитель, Главный бухгалтер, Руководитель кадровой службы, Кассир, Ответственный за налоговые регистры, Уполномоченный представитель, Исполнитель.

Физическое лицо - физическое лицо, являющееся ответственным для данной организации. Выбирается из справочника "Физические лица".

Должность - должность ответственного лица. Выбирается из справочника "Должности организаций".
