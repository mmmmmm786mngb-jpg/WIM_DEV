Документ "Назначение представителя счета депо"

Документ Назначение представителя счета депо предназначен для ввода в информационную базу сведений о присвоении статуса представителю счета депо на основании поручений депонентов (контрагентов) и расчета стоимости данной услуги депозитария, если в тарифном плане для этой услуги не указан тип тарифа Сумма за оборот.

Документ открывается для просмотра и редактирования из пункта меню "Документы/Административные операции/Назначение представителя счета депо"

Реквизиты

Основание - документ Поручение, на основании которого введен данный документ.

Владелец счета - депонент (контрагент) либо место хранения, которому требуется назначить представителя. При заполнении этого реквизита требуется сначала указать тип значения, а затем выбрать значение из справочника Депоненты/Контрагенты либо Места хранения.

Счет депо - счет депо, которому требуется назначить представителя.

Вид представителя счета депо - вид назначаемого представителя счета депо. Возможные значения: Оператор, Попечитель, Распорядитель.

Представитель счета депо - юридическое или физическое лицо, назначаемое представителем счета депо. Выбирается из справочника Депоненты/Контрагенты.

Данные доверенности - номер и период действия доверенности представителя счета депо.

Тарифный план - тарифный план, по которому следует рассчитывать стоимость услуги депозитария. Значение выбирается из справочника Тарифные планы.
