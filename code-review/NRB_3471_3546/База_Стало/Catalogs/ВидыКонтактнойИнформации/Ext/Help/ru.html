Справочник "Виды контактной информации"

Справочник Виды контактной информации предназначен для хранения информации обо всех видах контактной информации, использующихся в Типовом решении при вводе контактных данных организаций, депонентов (контрагентов), физических лиц, мест хранения и пользователей.

Справочник содержит набор предопределённых элементов, которые вводятся автоматически при первом запуске информационной базы Типового решения и недоступны для редактирования.

Справочник открывается для просмотра и редактирования из пункта меню "Справочники/ Виды контактной информации".
