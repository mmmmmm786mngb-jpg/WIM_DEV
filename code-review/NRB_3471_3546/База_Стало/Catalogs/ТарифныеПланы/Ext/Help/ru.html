Справочник "Тарифные планы"

Справочник "Тарифные планы" используется для хранения информации о тарифных планах на услуги депозитария.

Справочник открывается для просмотра и редактирования из пункта меню "Справочники/Тарифные планы".

В справочнике Тарифные планы требуется ввести не менее одного тарифного плана.

Информацию о стоимости услуг в тарифном плане можно посмотреть в подчиненном справочнике "Услуги тарифного плана".
