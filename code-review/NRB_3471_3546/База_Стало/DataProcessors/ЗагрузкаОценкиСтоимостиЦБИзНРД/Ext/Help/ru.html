Обработка «Загрузка оценки стоимости ЦБ из НРД»

Обработка предназначена для загрузки информации об оценке рыночной стоимости ценных бумаг для расчета платы за хранение из файлов, размещенных на странице НРД https://www.nsd.ru/ru/fees/estimation/.

 

Настройка обработки

Для загрузки оценочных котировок за 1 день необходимо выбрать дату загрузки и убрать флаг "Загрузить за месяц".

Для загрузки данных за месяц нужно установить флаг "Загрузить за месяц", а датой загрузки выбрать любое число загружаемого месяца.

 

Процесс загрузки данных

1) Нажать кнопку "Загрузить таблицу оценок ЦБ" - данные будут закачаны из Интернета, распакованы и закачаны в таблицу формы.

2) Нажать кнопку "Заполнить котировки из таблицы" - данные из таблицы будут помещены в регистр котировок цнных бумаг.

 

Контроль ошибок в базе данных

В процессе загрузки осуществляется поиск в базе данных кодов загрузки, у которых не существует соответствующей записи в справочнике ценных бумаг ("битые ссылки"), а также поиск бумаг, у которых нет соответствующего кода загрузки НРД, но есть код загрузки ММВБ.

При этом будут выводиться соответствующие сообщения на экран и в журнал регистрации.

Найденные ошибки не влияют на загрузку котировок всех остальных бумаг.
