Отчет "Операционный журнал расширенный"

Отчет Операционный журнал расширенный предназначен для автоматизированного формирования журнала, содержащего необходимую информацию обо всех выполненных и выполняемых депозитарием операциях, в том числе операций по счету депо. Если операция находится в процессе исполнения, часть полей операционного журнала счета депо может оставаться незаполненной.

Отчет открывается из пункта меню "Отчеты/Операционный журнал расширенный".
