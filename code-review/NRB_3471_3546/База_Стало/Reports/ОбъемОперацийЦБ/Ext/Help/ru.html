Отчет "Объем операций ЦБ"

Отчет Объем операций ЦБ предназначен для автоматизированного формирования отчетной формы "Количество и объёмы проведённых депозитарных операций с учитываемыми ценными бумагами" с учетом требований действующего  законодательства для ежеквартального предоставления в ФСФР.

Отчет открывается из пункта меню "Отчеты/ Объем операций ЦБ".
