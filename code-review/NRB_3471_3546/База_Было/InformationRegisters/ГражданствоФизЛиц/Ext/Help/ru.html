Регистр сведений "Гражданство физических лиц"

Регистр сведений "Гражданство физических лиц" используется для сохранения истории изменений гражданства физических лиц.

Реквизиты

Страна - страна, гражданином которой является данное физическое лицо. Значение выбирается из справочника Классификатор стран мира.

Действует с - дата актуальности записи регистра сведений Гражданство Физических лиц. По умолчанию в качестве даты актуальности устанавливается дата, указанная в параметрах информационной базы как Рабочая дата (Сервис/Параметры/ Общие).
