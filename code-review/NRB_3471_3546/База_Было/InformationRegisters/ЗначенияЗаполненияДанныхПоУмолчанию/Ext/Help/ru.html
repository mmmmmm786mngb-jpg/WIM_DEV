Регистр сведений "Значения заполнения данных по умолчанию"

В данном регистре хранятся значения реквизитов объектов метаданных, для заполнения по умолчанию

Регистр независимый - заполняется вручную. Периодичность - день

Для того, чтобы отменить использование заполнения объекта необходимо либо удалить все записи по объекта, либо последняя запись должна быть с признаком "Использование" = Ложь

Структура регистра:

  • Объект метаданных - имя и тип объекта метаданных. Маска заполнения: ТипМетаданных.ИмяОбъекта (например "Документ.ПлатежноеПоручение"). Заполнение обязательно
  • Настройка - имя настройки заполнения объекта метаданных. Заполнение обязательно
  • Условие - условие отбора записей. Произвольное значение (простой тип или ссылка) для отбора записей регистра
  • Использование - признак использования данной настройки для заполнения
  • Значения - значения реквизитов объекта для заполнения по умолчанию (хранилище значений)
