Отчет "Сведения о количестве счетов депо"

Отчет Сведения о количестве счетов депо предназначен для автоматизированного формирования отчетной формы "Сведения о количестве счетов депо депонентов" с учетом требований действующего законодательства для ежеквартального предоставления в ФСФР.

Отчет открывается из пункта меню "Отчеты/ Сведения о количестве счетов депо".
