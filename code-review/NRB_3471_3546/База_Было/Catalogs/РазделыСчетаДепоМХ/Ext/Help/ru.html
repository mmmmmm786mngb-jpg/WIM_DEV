Справочник "Разделы счета Депо МХ"

Справочник Разделы счета Депо МХ предназначен для хранения основных характеристик разделов счетов Депо мест хранения.

Справочник подчинен справочнику Счета Депо МХ.

Справочник можно открыть для просмотра и редактирования из формы элемента справочника Места хранения на закладке "Учет ЦБ".

Реквизиты

Входящий номер - входящий номер поручения на открытие раздела счета депо МХ

Операция открытия раздела счета - Документ Открытие раздела счета депо, на основании которого открыт раздел счета депо

Дата открытия счета - дата открытия раздела счета депо МХ

Статус счета депо - статус раздела счета депо МХ. Возможные значения: Открыт, Закрыт, Не открыт

Дата закрытия счета - дата закрытия раздела счета депо МХ
