Справочник "Счета Депо МХ"

Справочник Счета Депо МХ предназначен для учета прав собственности места хранения на ценные бумаги.

Справочник можно открыть для просмотра и редактирования из формы элемента справочника-владельца Места Хранения на закладке "Учет ЦБ".

Реквизиты

Код - код счета депо МХ

Тип счета - тип счета депо МХ. Возможные значения: Счет депо Собственника, Доверительного управляющего, Залогодержателя, Номинального держателя

Номер договора - номер депозитарного договора места хранения

Дата договора - дата заключения депозитарного договора с местом хранения

Основной раздел - основной раздел счета депо МХ. Выбирается из справочника Разделы счета депо МХ

Дата открытия счета - дата открытия счета депо МХ

Статус счета - статус счета депо МХ. Возможные значения: Открыт, Закрыт, Не открыт

Дата закрытия счета - дата закрытия счета депо МХ

Операция закрытия счета депо - документ Закрытие счета депо, в соответствии с которым закрыт счет депо
