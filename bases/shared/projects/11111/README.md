# Проект 11111

Демонстрационная внешняя обработка с управляемой формой: реквизит типа Строка и команда вывода значения.

## Состав

- **src/** — XML-исходники внешней обработки `Обработка11111` (сборка в `.epf` через конфигуратор или `/epf-build` из cc-1c-skills)
- **Расширения/Ext11111_WimDu** — расширение для **Wim_Du**: отчет СКД по остаткам регистра **Хозрасчетный** в разрезе **ДоговорДУ** с отбором по счету (см. `Расширения/README.md`)
- **Документация/** — описание проекта
- **Тестирование/** — сценарии и отчеты
- **Скрипты/** — автоматизация

## Базы

Список баз — в `bases.txt` (пути к исходникам: `../../<база>/source-path.txt`).

## Сборка EPF

Из каталога `tools/cc-1c-skills` (при необходимости поправьте путь к `1cv8.exe` в настройках навыка):

```text
python scripts/switch.py cursor
```

Далее в Cursor вызовите навык сборки или вручную загрузите XML в конфигуратор и сохраните обработку как `.epf`.

## Поведение формы

1. Реквизит формы **ВведеннаяСтрока** (строка неограниченной длины).
2. Команда **ПоказатьВведеннуюСтроку** — кнопка на форме; вызывает `Сообщить(ВведеннаяСтрока)`.

Платформа: не ниже 8.3.21.
