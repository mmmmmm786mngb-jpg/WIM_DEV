# Проект example (shared)

Шаблон проекта по нескольким базам. Переименуйте папку под название проекта.

## Какие базы использует проект

Файл **bases.txt** в этой папке — список баз (по одной в строке). Путь к исходникам каждой базы: **../../<база>/source-path.txt**.

Пример bases.txt:
```
SBB_Broker
Wim_Du
```

## Структура проекта

- **bases.txt** — список баз (опционально, для скриптов)
- **Расширения/** — доработки, расширения
- **Документация/** — ТЗ, спецификации
- **Тестирование/** — сценарии тестов, отчёты (reports/), Python COM
- **Скрипты/** — автоматизация; пути к базам читать из ../../<база>/source-path.txt

## Требования

- Платформа 1С:Предприятие не ниже 8.3.21
- EDT или конфигуратор
