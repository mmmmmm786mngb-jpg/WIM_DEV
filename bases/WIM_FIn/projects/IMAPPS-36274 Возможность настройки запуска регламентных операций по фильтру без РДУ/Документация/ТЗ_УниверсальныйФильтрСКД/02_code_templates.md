# Готовые тексты BSL для вставки

Файл содержит шаблоны кода для разработчика.
Перед вставкой проверить имена реквизитов формы и доступность типовых форм компоновщика в текущей базе.

**Реквизит формы `НастройкиСКД` (дерево отбора СКД на управляемой форме):** в метаданных тип должен быть **`КомпоновщикНастроекКомпоновкиДанных`**, иначе привязка `НастройкиСКД.Settings.Filter` не будет работать штатно. В `ХранилищеЗначения` по-прежнему сохраняют `НастройкиКомпоновкиДанных` через `НастройкиСКД.ПолучитьНастройки()`. Подробнее: `04_developer_brief.html`.

## 1) `Catalogs/НастройкиЗакрытияПортфелейПоРасписанию/Ext/ObjectModule.bsl`

Вставить в конец модуля (под `#КонецОбласти` обработчиков).

```bsl
// Описание: Сохраняет настройки универсального СКД-фильтра в реквизит элемента
//
// Параметры:
// НастройкиКомпоновки - НастройкиКомпоновкиДанных - объект настроек СКД для сохранения
//
Процедура СохранитьНастройкиУниверсальногоФильтраСКД(НастройкиКомпоновки) Экспорт
	
	Если НастройкиКомпоновки = Неопределено Тогда
		НастройкиУниверсальногоФильтраСКД = Неопределено;
		ПредставлениеУниверсальногоФильтраСКД = "";
		Возврат;
	КонецЕсли;
	
	НастройкиУниверсальногоФильтраСКД = Новый ХранилищеЗначения(НастройкиКомпоновки);
	ПредставлениеУниверсальногоФильтраСКД = СформироватьПредставлениеУниверсальногоФильтраСКД(НастройкиКомпоновки);
	
КонецПроцедуры

// Описание: Возвращает сохраненные настройки универсального СКД-фильтра
//
// Возвращаемое значение:
// НастройкиКомпоновкиДанных, Неопределено
//
Функция ПолучитьНастройкиУниверсальногоФильтраСКД() Экспорт
	
	Если НЕ ЗначениеЗаполнено(НастройкиУниверсальногоФильтраСКД) Тогда
		Возврат Неопределено;
	КонецЕсли;
	
	Попытка
		Возврат НастройкиУниверсальногоФильтраСКД.Получить();
	Исключение
		ЗаписьЖурналаРегистрации(
			"УниверсальныйФильтрСКД",
			УровеньЖурналаРегистрации.Ошибка,
			,
			,
			"Не удалось прочитать сохраненные настройки СКД: " + ОписаниеОшибки());
	КонецПопытки;
	
	Возврат Неопределено;
	
КонецФункции

// Описание: Очищает настройки и представление универсального фильтра СКД
//
Процедура СброситьНастройкиУниверсальногоФильтраСКД() Экспорт
	НастройкиУниверсальногоФильтраСКД = Неопределено;
	ПредставлениеУниверсальногоФильтраСКД = "";
КонецПроцедуры

// Описание: Формирует текстовое представление включенных отборов СКД
//
// Параметры:
// НастройкиКомпоновки - НастройкиКомпоновкиДанных
//
// Возвращаемое значение:
// Строка - многострочное описание фильтров
//
Функция СформироватьПредставлениеУниверсальногоФильтраСКД(НастройкиКомпоновки) Экспорт
	
	Если НастройкиКомпоновки = Неопределено Тогда
		Возврат "";
	КонецЕсли;
	
	Текст = Новый ТекстовыйДокумент;
	Текст.ДобавитьСтроку("Универсальный фильтр СКД:");
	
	Если НЕ ЗначениеЗаполнено(НастройкиКомпоновки.Отбор) Тогда
		Текст.ДобавитьСтроку("- без условий");
		Возврат СокрЛП(Текст.ПолучитьТекст());
	КонецЕсли;
	
	Для каждого ЭлементОтбора Из НастройкиКомпоновки.Отбор.Элементы Цикл
		Если НЕ ЭлементОтбора.Использование Тогда
			Продолжить;
		КонецЕсли;
		
		Текст.ДобавитьСтроку(
			СтрШаблон("- %1 %2 %3",
				Строка(ЭлементОтбора.ЛевоеЗначение),
				Строка(ЭлементОтбора.ВидСравнения),
				Строка(ЭлементОтбора.ПравоеЗначение)));
	КонецЦикла;
	
	Возврат СокрЛП(Текст.ПолучитьТекст());
	
КонецФункции
```

## 2) `Catalogs/Портфели/Ext/ManagerModule.bsl`

Вставить рядом с `СписокПортфелейПоДинамическомуСписку`.

```bsl
// Описание: Возвращает массив портфелей по сохраненной пользовательской настройке СКД
//
// Параметры:
// ДатаПериода - Дата - дата расчета
// НастройкиУниверсальногоФильтраСКД - ХранилищеЗначения, НастройкиКомпоновкиДанных, Неопределено
//
// Возвращаемое значение:
// Массив, Неопределено
//
Функция СписокПортфелейПоСохраненнойНастройкеСКД(ДатаПериода, НастройкиУниверсальногоФильтраСКД) Экспорт
	
	Если НЕ ЗначениеЗаполнено(НастройкиУниверсальногоФильтраСКД) Тогда
		Возврат Неопределено;
	КонецЕсли;
	
	Попытка
		Если ТипЗнч(НастройкиУниверсальногоФильтраСКД) = Тип("ХранилищеЗначения") Тогда
			НастройкиПользователя = НастройкиУниверсальногоФильтраСКД.Получить();
		Иначе
			НастройкиПользователя = НастройкиУниверсальногоФильтраСКД;
		КонецЕсли;
	Исключение
		ЗаписьЖурналаРегистрации("УниверсальныйФильтрСКД", УровеньЖурналаРегистрации.Ошибка, , ,
			"Ошибка чтения хранилища СКД: " + ОписаниеОшибки());
		Возврат Неопределено;
	КонецПопытки;
	
	Если НастройкиПользователя = Неопределено Тогда
		Возврат Неопределено;
	КонецЕсли;
	
	ТаблицаЗначений = Новый ТаблицаЗначений;
	СКД = Справочники.Портфели.ПолучитьМакет("ДействующиеПортфели");
	
	КомпоновщикНастроек = Новый КомпоновщикНастроекКомпоновкиДанных;
	КомпоновщикНастроек.Инициализировать(Новый ИсточникДоступныхНастроекКомпоновкиДанных(СКД));
	КомпоновщикНастроек.ЗагрузитьНастройки(СКД.НастройкиПоУмолчанию);
	КомпоновщикНастроек.ЗагрузитьНастройки(НастройкиПользователя);
	
	НастройкиКомпоновщика = КомпоновщикНастроек.ПолучитьНастройки();
	Параметр = НастройкиКомпоновщика.ПараметрыДанных.Элементы.Найти("ДатаПериода");
	Если Параметр <> Неопределено Тогда
		Параметр.Значение = ДатаПериода;
		Параметр.Использование = Истина;
	КонецЕсли;
	
	КомпоновщикМакета = Новый КомпоновщикМакетаКомпоновкиДанных;
	МакетКомпоновкиДанных = КомпоновщикМакета.Выполнить(
		СКД,
		НастройкиКомпоновщика,
		,
		,
		Тип("ГенераторМакетаКомпоновкиДанныхДляКоллекцииЗначений"));
	
	ПроцессорКомпоновкиДанных = Новый ПроцессорКомпоновкиДанных;
	ПроцессорКомпоновкиДанных.Инициализировать(МакетКомпоновкиДанных);
	
	ПроцессорВывода = Новый ПроцессорВыводаРезультатаКомпоновкиДанныхВКоллекциюЗначений;
	ПроцессорВывода.УстановитьОбъект(ТаблицаЗначений);
	ПроцессорВывода.Вывести(ПроцессорКомпоновкиДанных);
	
	Возврат ТаблицаЗначений.ВыгрузитьКолонку("Ссылка");
	
КонецФункции
```

## 3) `Catalogs/НастройкиЗакрытияПортфелейПоРасписанию/Forms/ФормаЭлемента/Ext/Form/Module.bsl`

Добавить команды формы (кнопки) и обработчики.

```bsl
&НаКлиенте
Процедура НастроитьУниверсальныйФильтрСКД(Команда)
	НастроитьУниверсальныйФильтрСКДНаСервере();
	УправлениеФормой();
КонецПроцедуры

&НаКлиенте
Процедура ОчиститьУниверсальныйФильтрСКД(Команда)
	Объект.НастройкиУниверсальногоФильтраСКД = Неопределено;
	Объект.ПредставлениеУниверсальногоФильтраСКД = "";
	УправлениеФормой();
КонецПроцедуры

&НаСервере
Процедура НастроитьУниверсальныйФильтрСКДНаСервере()
	
	Если НЕ Объект.ИспользоватьУниверсальныйФильтрСКД Тогда
		Возврат;
	КонецЕсли;
	
	СКД = Справочники.Портфели.ПолучитьМакет("ДействующиеПортфели");
	Компоновщик = Новый КомпоновщикНастроекКомпоновкиДанных;
	Компоновщик.Инициализировать(Новый ИсточникДоступныхНастроекКомпоновкиДанных(СКД));
	Компоновщик.ЗагрузитьНастройки(СКД.НастройкиПоУмолчанию);
	
	Если ЗначениеЗаполнено(Объект.НастройкиУниверсальногоФильтраСКД) Тогда
		Попытка
			Компоновщик.ЗагрузитьНастройки(Объект.НастройкиУниверсальногоФильтраСКД.Получить());
		Исключение
			ЗаписьЖурналаРегистрации("УниверсальныйФильтрСКД", УровеньЖурналаРегистрации.Ошибка, , ,
				"Ошибка загрузки сохраненных настроек: " + ОписаниеОшибки());
		КонецПопытки;
	КонецЕсли;
	
	// ВАЖНО: ниже использовать типовую форму настройки компоновщика, принятую в вашей конфигурации.
	// Пример вызова зависит от реализованного в проекте шаблона.
	// После закрытия формы получить НастройкиКомпоновки и сохранить:
	//
	// Объект.НастройкиУниверсальногоФильтраСКД = Новый ХранилищеЗначения(Компоновщик.ПолучитьНастройки());
	// Объект.ПредставлениеУниверсальногоФильтраСКД =
	//     Справочники.НастройкиЗакрытияПортфелейПоРасписанию.СформироватьПредставлениеУниверсальногоФильтраСКД(Компоновщик.ПолучитьНастройки());
	
КонецПроцедуры
```

И в существующей `УправлениеФормой()` добавить:

```bsl
Элементы.НастроитьУниверсальныйФильтрСКД.Доступность = Объект.ИспользоватьУниверсальныйФильтрСКД;
Элементы.ОчиститьУниверсальныйФильтрСКД.Доступность = Объект.ИспользоватьУниверсальныйФильтрСКД;
Элементы.ПредставлениеУниверсальногоФильтраСКД.Видимость = Объект.ИспользоватьУниверсальныйФильтрСКД;
```

## 4) `DataProcessors/ГрупповоеВыполнениеЗакрытияПериодов/Ext/ObjectModule.bsl`

### 4.1. Доработка `ЗаполнитьРеквизитыПоНастройкеРегулярногоЗакрытия`

Добавить строки сразу после присваивания `СоздаватьОбщиеОперации`:

```bsl
	ЭтотОбъект.ИспользоватьУниверсальныйФильтрСКД = НастройкаРегулярногоЗакрытия.ИспользоватьУниверсальныйФильтрСКД;
	ЭтотОбъект.НастройкиУниверсальногоФильтраСКД = НастройкаРегулярногоЗакрытия.НастройкиУниверсальногоФильтраСКД;
```

### 4.2. Добавить функцию пересечения

```bsl
// Описание: Дополнительно отфильтровывает список портфелей по сохраненной СКД-настройке
//
// Параметры:
// ДатаЗакрытия - Дата - дата расчета
// МассивПортфелейИсходный - Массив - базовый список после штатных фильтров
//
// Возвращаемое значение:
// Массив - итоговый список после пересечения
//
Функция ОтфильтроватьМассивПортфелейУниверсальнымФильтромСКД(ДатаЗакрытия, МассивПортфелейИсходный)
	
	Если НЕ ЭтотОбъект.ИспользоватьУниверсальныйФильтрСКД Тогда
		Возврат МассивПортфелейИсходный;
	КонецЕсли;
	
	Если НЕ ЗначениеЗаполнено(ЭтотОбъект.НастройкиУниверсальногоФильтраСКД) Тогда
		Возврат МассивПортфелейИсходный;
	КонецЕсли;
	
	Если НЕ ЗначениеЗаполнено(МассивПортфелейИсходный) Тогда
		Возврат Новый Массив;
	КонецЕсли;
	
	МассивПоСКД = Справочники.Портфели.СписокПортфелейПоСохраненнойНастройкеСКД(
		ДатаЗакрытия,
		ЭтотОбъект.НастройкиУниверсальногоФильтраСКД);
	
	Если НЕ ЗначениеЗаполнено(МассивПоСКД) Тогда
		Возврат МассивПортфелейИсходный;
	КонецЕсли;
	
	ИндексСКД = Новый Соответствие;
	Для каждого ПортфельСКД Из МассивПоСКД Цикл
		ИндексСКД.Вставить(ПортфельСКД, Истина);
	КонецЦикла;
	
	Результат = Новый Массив;
	Для каждого ПортфельИсходный Из МассивПортфелейИсходный Цикл
		Если ИндексСКД.Получить(ПортфельИсходный) = Истина Тогда
			Результат.Добавить(ПортфельИсходный);
		КонецЕсли;
	КонецЦикла;
	
	Возврат Результат;
	
КонецФункции
```

### 4.3. Доработка `ЗаполнитьСписокПортфелей`

После строки:

```bsl
	ДействующиеПортфели = Справочники.Портфели.ДействующиеПортфели(ДатаЗакрытия, СписокТипов, СписокПортфелейПоСтратегиям);
```

добавить:

```bsl
	ДействующиеПортфели = ОтфильтроватьМассивПортфелейУниверсальнымФильтромСКД(ДатаЗакрытия, ДействующиеПортфели);
```

### 4.4. Доработка веток по границам

В `ЗаполнитьПериодыПортфелейПоГраницам` и `ЗаполнитьПериодыПортфелейПоВарианту_ДНГ_ДКР`:
- когда формируется `ДействующиеПортфели` на текущий день, сразу применить
`ОтфильтроватьМассивПортфелейУниверсальнымФильтромСКД(День, ДействующиеПортфели)`.

Это обеспечит одинаковую семантику для всех вариантов периода.

## 6) Что проверить после вставки

1. Компиляция модулей и форм без ошибок типов.
2. Форма элемента справочника:
   - настройки открываются,
   - представление фильтра меняется,
   - запись элемента работает.
3. Регламент:
   - при включенном фильтре итоговый `ПериодыПортфелей` уменьшается согласно условиям СКД.
4. При выключенном флаге функционал не влияет на старое поведение.
