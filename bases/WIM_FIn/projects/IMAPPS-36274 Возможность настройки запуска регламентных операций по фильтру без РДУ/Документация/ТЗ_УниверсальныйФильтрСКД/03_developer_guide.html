Добавить реквизит объекта НастройкиУниверсальногоФильтраПортфелейСКД в справочник
XML метаданные
Что делаем
Добавляем в справочник НастройкиЗакрытияПортфелейПоРасписанию один новый реквизит типа ХранилищеЗначения для хранения сериализованных настроек СКД.

Файл для редактирования

▶ Catalogs/НастройкиЗакрытияПортфелейПоРасписанию.xml

Где искать место вставки

В файле .xml найдите блок <Attributes>. Внутри него уже есть реквизиты: ПоВсемДействующимПортфелям, ТипыКлиентов и другие. Добавьте новый реквизит после последнего существующего реквизита, перед закрывающим тегом </Attributes>.

Про UUID
Для каждого нового реквизита нужен уникальный UUID. Сгенерируйте его в 1C EDT (при создании реквизита через дизайнер) или вручную любым UUID-генератором. Формат: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx.

Результат
В таблице справочника появится колонка НастройкиУниверсальногоФильтраПортфелейСКД типа ХранилищеЗначения. Там будут храниться XML-настройки компоновщика.

Как использовать EDT вместо XML

  1. Откройте объект НастройкиЗакрытияПортфелейПоРасписанию в дереве метаданных.
  2. Вкладка Данные → Реквизиты → кнопка "Добавить".
  3. Имя: НастройкиУниверсальногоФильтраПортфелейСКД, Тип: ХранилищеЗначения.
  4. Синоним (рус.): "Фильтр портфелей (СКД)".
  5. Сохраните. EDT сам запишет XML с правильным UUID.

Добавить реквизит формы КомпоновщикУниверсальногоФильтраПортфелейСКД и кнопку в Form.xml
UI / Form.xml
Что делаем
В форме элемента справочника (ФормаЭлемента) добавляем:
1. Реквизит формы КомпоновщикУниверсальногоФильтраПортфелейСКД типа КомпоновщикНастроекКомпоновкиДанных — для штатного дерева отбора СКД на форме (в хранилище справочника по-прежнему сохраняют НастройкиКомпоновкиДанных через ПолучитьНастройки()).
2. Реквизит формы ПредставлениеФильтраСКД (строка) — для отображения текущего фильтра пользователю (при варианте с отдельной формой настроек; при встроенной таблице отбора может не понадобиться).
3. Группу на форме с кнопкой "Настроить фильтр портфелей" и полем-декорацией для показа фильтра или таблицей КомпоновщикУниверсальногоФильтраПортфелейСКД.Settings.Filter (актуальный вариант без отдельного окна).
Тип реквизита КомпоновщикУниверсальногоФильтраПортфелейСКД в метаданных
Чтобы дерево отбора на управляемой форме работало штатно, в свойствах реквизита формы нужно указать тип КомпоновщикНастроекКомпоновкиДанных (в XML: dcsset:SettingsComposer). Реквизит с типом НастройкиКомпоновкиДанных (dcscor:DataCompositionSettings) для привязки DataPath = КомпоновщикУниверсальногоФильтраПортфелейСКД.Settings.Filter не используют.

Как это устроено: открытие компоновщика в 1С

В 1С нет прямого объекта "ДиалогКомпоновщикаНастроек". Стандартный способ:

  1. Создать объект КомпоновщикНастроекКомпоновкиДанных.
  2. Инициализировать его СКД-схемой.
  3. Загрузить текущие настройки (или настройки по умолчанию).
  4. Открыть форму "ОбщаяФорма.НастройкиКомпоновкиДанных" (если она есть в конфигурации) или стандартную платформенную форму через ОткрытьФорму("ОбщаяФорма.НастройкиКомпоновкиДанных", ...).
Если в конфигурации нет общей формы настроек СКД
Используем альтернативный способ: открываем форму через НастройкиКомпоновщика.ОткрытьФормуНастроек(). Однако этот метод не доступен в тонком клиенте. Рекомендуемый подход — передать компоновщик как параметр в специально созданную вспомогательную форму. Ниже приведён рабочий вариант.

Серверная функция получения схемы и настроек

Form/Module.bsl — раздел #Область СлужебныеПроцедурыИФункции 
// Описание: Возвращает схему СКД для фильтра портфелей и текущие сохранённые настройки.
//           Если настройки ещё не сохранены — возвращает настройки по умолчанию.
//
// Возвращаемое значение:
//   Структура - поля: Схема (СхемаКомпоновкиДанных), Настройки (НастройкиКомпоновкиДанных)
&НаСервере
Функция ПолучитьСхемуИНастройкиФильтраПортфелейНаСервере()
    
    // Берём СКД-схему из макета "ДействующиеПортфели" справочника Портфели
    Схема = Справочники.Портфели.ПолучитьМакет("ДействующиеПортфели");
    
    // Пробуем прочитать сохранённые настройки из реквизита объекта
    СохранённыеНастройки = Неопределено;
    
    ХранилищеЗначения = Объект.НастройкиУниверсальногоФильтраПортфелейСКД;
    Если ТипЗнч(ХранилищеЗначения) = Тип("ХранилищеЗначения") Тогда
        СохранённыеНастройки = ХранилищеЗначения.Получить();
    КонецЕсли;
    
    Если ТипЗнч(СохранённыеНастройки) <> Тип("НастройкиКомпоновкиДанных") Тогда
        // Первый запуск — используем настройки по умолчанию из схемы
        СохранённыеНастройки = Схема.НастройкиПоУмолчанию;
    КонецЕсли;
    
    Результат = Новый Структура("Схема, Настройки", Схема, СохранённыеНастройки);
    Возврат Результат;
