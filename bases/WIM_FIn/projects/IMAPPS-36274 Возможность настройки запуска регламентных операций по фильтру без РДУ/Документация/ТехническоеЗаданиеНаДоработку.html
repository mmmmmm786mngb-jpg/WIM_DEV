Техническое задание на доработку

Задача: IMAPPS-36274 Возможность настройки запуска регламентных операций по фильтру без РДУ

База: WIM_FIn (ДУ редакция 2)

Дата: 03.04.2026

1. Цель доработки

Добавить пользовательский отбор СКД для портфелей без изменения базового алгоритма закрытия периода.

Новый отбор работает как пост-фильтр: на вход получает список портфелей из старого механизма, на выходе возвращает отфильтрованный список.

Если настройки СКД пустые, должен возвращаться исходный список без изменений.

5. Обязательные технические правила

  • Расширить макет Справочники.Портфели.Templates.ДействующиеПортфели
  • Добавить в СКД параметр СписокПортфелей
  • Сохранять настройки компоновщика через ХранилищеЗначения
  • Если отбор в настройках СКД отсутствует, функция фильтра возвращает исходный список
  • Параметр даты в новой функции не использовать

6. Референсный паттерн (демо конфигурация)

Эталон в демо: справочник с отбором СКД, хранением настроек в ХранилищеЗначения и макетом схемы компоновки.
