2. Что изменилось в макете

2.1 Новые секции

  • Заголовок2 - секция методики расчета платы за управление.
  • Подвал2 - дополнительный блок подписей.

2.2 Новые/измененные параметры

Область Параметры Комментарий
Заголовок2 ПериодРасчета, ПлатаЗаУправление, П000 Второй заголовок с блоком MF
ТелоРасчета П021Строка, ВознаграждениеЗаУспех, ИтоговоеВознаграждениеЗаУспех Новые параметры для SF-блока
ПодвалСЧА СЧА Важно: имя параметра стало СЧА (не Сча)

2.3 Формулы/тексты, которые были уточнены

  • Ставка SF: пороги 250/500 млн и ставки 6/5.25/5.
  • Условия по SF: ветка <= 0 для нулевого результата.
  • Переход к обозначениям новой версии макета (включая Стр. MF / Стр. SF).

3. Что изменилось в коде и почему

3.1 Печать по новым областям

В функции СправкаРасчет() добавлено получение и вывод Заголовок2/Подвал2, чтобы новая структура макета реально выводилась в табличный документ.

// IMAPPS-35720: новый макет содержит дополнительную секцию методики платы за управление.
ОбластьЗаголовок2 = Макет.ПолучитьОбласть("Заголовок2");
// IMAPPS-35720: новый макет содержит дополнительный блок подписей Подвал2.
ОбластьПодвал2    = Макет.ПолучитьОбласть("Подвал2");

...

Если ОбластьЗаголовок2.Параметры.Свойство("ПлатаЗаУправление") Тогда
    ОбластьЗаголовок2.Параметры.ПлатаЗаУправление = Формат(ПоказателиСправки.П000, "ЧДЦ=2; ЧН=0,00");
КонецЕсли;
Если ОбластьЗаголовок2.Параметры.Свойство("П000") Тогда
    ОбластьЗаголовок2.Параметры.П000 = ПоказателиСправки.П000;
КонецЕсли;
ТабДокументПодвал.Вывести(ОбластьЗаголовок2);
ТабДокументПодвал.Вывести(ОбластьПодвал2);

3.2 Поддержка новых параметров SF в теле

Добавлены поля структуры ПоказателиСправки, чтобы ЗаполнитьЗначенияСвойств() корректно заполнял новые параметры макета.

ПоказателиСправки.Вставить("П021Строка", "");
ПоказателиСправки.Вставить("ВознаграждениеЗаУспех", 0);
ПоказателиСправки.Вставить("ИтоговоеВознаграждениеЗаУспех", 0);

...

ПоказателиСправки.П021Строка = Формат(ПроцентВознаграждения, "ЧДЦ=2; ЧН=0,00") + "%";
ПоказателиСправки.ВознаграждениеЗаУспех = ПоказателиСправки.П022;
ПоказателиСправки.ИтоговоеВознаграждениеЗаУспех = ПоказателиСправки.П023;

3.3 Исправление имени параметра в блоке СЧА

В новом макете параметр называется СЧА, поэтому в коде вывод изменен на это имя.

// IMAPPS-35720: в новом макете параметр называется "СЧА".
ОбластьПодвалСЧА.Параметры.СЧА = Формат(ПоказателиСправки.СтруктураСЧА.СЧА, "ЧДЦ=2");

3.4 Расчетная часть (ключевые изменения относительно типовой)

  • Ставка SF выбирается по таблице коэффициентов от СЧА на конец периода.
  • РасчетБРСА стал самодостаточным (без зависимости от внешнего состояния).
  • Сохранены комментарии с номером задачи IMAPPS-35720 в измененных местах.
