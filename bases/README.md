## Структура каждой базы

```
bases/<база>/
  source-path.txt   # Одна строка: путь к каталогу с реальной базой
  README.md
  Проекты/          # Здесь создаются проекты
    example/        # Шаблон (только в SBB_Broker)
    <ваш-проект>/
      Расширения/
      Документация/
      Тестирование/
      Скрипты/
```

Проекты читают путь к источнику из **../source-path.txt** (относительно папки проекта).

## Общие проекты (несколько баз)

Папка **Общие/** — для проектов, которые работают с двумя и более базами (интеграции, общие отчёты и т.п.).

```
bases/Общие/
  README.md
  Проекты/
    <проект>/
      bases.txt      # Список баз (по одной в строке), опционально
      Расширения/
      Документация/
      Тестирование/
      Скрипты/
```

Пути к исходникам: **../../<база>/source-path.txt** (например, ../../SBB_Broker/source-path.txt). В проекте можно указать список баз в **bases.txt** — тогда скрипты смогут по нему подставлять пути.
