# Projects on base SBB_Broker

Здесь создаются проекты. Каждый проект — папка с подпапками: Расширения, Документация, Тестирование, Скрипты.

Путь к исходникам базовой конфигурации: в файле **../source-path.txt** (от корня базы; из папки проекта — ../../source-path.txt) (не копируем базу, используем путь).

Шаблон: папка **example/** — скопируйте под новый проект.
