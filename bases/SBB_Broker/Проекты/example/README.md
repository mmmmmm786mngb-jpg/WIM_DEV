# Проект example

Шаблон проекта на базе SBB_Broker. Переименуйте папку под название проекта.

## Источник кода базы

Путь к каталогу с исходниками базовой конфигурации (не изменять) — в файле **../../source-path.txt**.  
Скрипты и инструменты должны читать путь оттуда.

## Структура проекта

- **Расширения/** — доработки: расширения конфигурации
- **Документация/** — ТЗ, спецификации, описание доработок
- **Тестирование/** — сценарии тестов, отчёты (reports/), Python COM
- **Скрипты/** — выгрузка/загрузка, миграции, интеграции

## Требования

- Платформа 1С:Предприятие не ниже 8.3.21
- EDT или конфигуратор
