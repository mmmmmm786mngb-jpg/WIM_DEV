# Проект pif_test_account_balance_report

Тестовый проект для внешнего отчета ПИФ: остатки по выбранному счету плана счетов.

## Структура

- `Расширения/` - исходники внешнего отчета
- `Документация/` - описание доработки
- `Тестирование/` - сценарии проверки и отчеты
- `Скрипты/` - служебные скрипты
