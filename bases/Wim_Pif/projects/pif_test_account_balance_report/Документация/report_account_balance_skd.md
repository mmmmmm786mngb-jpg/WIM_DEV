# Отчет по остаткам по счету (СКД)

## Что сделано

- Создан внешний отчет `ОстАткиПоСчету`.
- Источник данных: `РегистрБухгалтерии.Управленческий.Остатки`.
- Параметры пользователя: `Счет` и `ДатаОстатков`.
- `ДатаОстатков` инициализируется выражением `КонецДня(ТекущаяДата())`.

## Путь к файлам

- Исходники: `Расширения/ОстАткиПоСчету/src`
- Скомпилированный файл: `Расширения/ОстАткиПоСчету/build/ОстАткиПоСчету.erf`
