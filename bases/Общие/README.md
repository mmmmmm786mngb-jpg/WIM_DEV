# Общие проекты

Проекты, которые работают **с несколькими базами** (интеграции, общие отчёты, перенос данных и т.п.).

Пути к исходникам баз не копируются: для каждой базы используйте файл **source-path.txt** из соответствующей папки в bases/, например:
- `../SBB_Broker/source-path.txt`
- `../Wim_Du/source-path.txt`

## Структура

```
bases/Общие/
  README.md
  Проекты/
    <проект>/
      bases.txt      # Опционально: список баз (по одной в строку), с которыми работает проект
      Расширения/
      Документация/
      Тестирование/
      Скрипты/
```

Шаблон: **Проекты/example/**.
