# Общие проекты (несколько баз)

Здесь создаются проекты, которые используют две и более базы.  
Пути к исходникам: в папках баз, файл **source-path.txt** (например, ../../SBB_Broker/source-path.txt).

В проекте можно добавить **bases.txt** — по одной базе в строку (SBB_Broker, Wim_Du и т.д.), чтобы скрипты знали, с какими базами работать.

Шаблон: папка **example/**.
