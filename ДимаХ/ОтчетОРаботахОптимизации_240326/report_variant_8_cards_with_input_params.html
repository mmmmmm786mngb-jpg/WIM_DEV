Выводы

  • По большинству сценариев из Excel подтверждается существенное ускорение после оптимизаций, особенно в исполнении сделок и регламентах.
  • Наибольший подтвержденный эффект сохраняется в проведении сделок и обменах по сопоставимым сценариям.
  • Для блока "Загрузка сделок" выполнена нормализация: 20 мин на 900 сделок эквивалентно примерно 667 мин на 30000 (линейная оценка), текущее значение 50 мин.
  • Для утренних и вечерних регламентов baseline был на 15000 договоров, а текущие замеры на 3000; в отчете применена линейная нормализация к единому объему 3000.
  • Для ФИН исполнения baseline в Excel дан по общему циклу, поэтому разбиение на этапы открытие/закрытие стоит подтверждать отдельными детальными замерами.
  • По блокам со статусом "Вендор передал в UAT" фактический финальный эффект нужно подтвердить по итогам UAT и после выката.
